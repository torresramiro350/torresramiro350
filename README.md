### About me

I'm an astrophysicist (PhD, Astronomy — Shanghai Jiao Tong University) transitioning into research software engineering. Since 2020 I've maintained **Hal**, the HAWC likelihood-analysis plugin referenced in nearly all HAWC Galactic-analysis publications, built within the open-source **3ML** framework. I ship scientific software used beyond my own collaboration via PyPI/conda, and build Python-based analysis pipelines across a 30+ institution international collaboration.

- 🔭 Maintainer of **Hal** — [threeML/hawc_hal](https://github.com/threeML/hawc_hal)
- 🧩 Contributor to **astromodels** — look-up table interpolation class for continuous likelihood modeling
- 🌎 Work spans the **HAWC**, **SWGO**, **Fermi-LAT**, and **VERITAS** collaborations
- 🛠️ Currently focused on HPC-compatible refactors, CI/CD, and reproducible packaging for scientific pipelines
- 📄 11 publications (ADS / ORCID) — lead author on pulsar halo & cosmic-ray diffusion work
